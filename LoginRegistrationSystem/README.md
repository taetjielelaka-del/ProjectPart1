# LoginRegistrationSystem

A simple NetBeans-friendly Java Swing application for user registration and login.

## Features

- Register a new account with full name, username, and password
- Prevent duplicate usernames
- Log in with saved credentials
- Store users locally in `data/users.txt`

## Open In NetBeans

1. Open NetBeans.
2. Choose **File > Open Project**.
3. Select the `LoginRegistrationSystem` folder.
4. Run the project.
